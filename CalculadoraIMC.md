# Replicação de Projeto Tutorial (Calculadora de IMC) - Desenvolvimento Mobile

## Objetivo
Assistir às aulas e replicar fielmente o aplicativo de Calculadora de IMC desenvolvido nos vídeos:

- Parte 1 (Layout): [Android Studio: Criando um App de Calculadora de IMC com Jetpack Compose - Parte 1🔥🔥](https://www.youtube.com/watch?v=G_k0GAS0J9Y)
- Parte 2 (Lógica): [Android Studio: Criando um App de Calculadora de IMC com Jetpack Compose - Parte 2🔥🔥](https://www.youtube.com/watch?v=m9P7hYD50KM)

Critérios de Entrega:
- Código Fonte: O projeto.
- Funcionalidade: O app deve funcionar.
- Interface: Implementar o Scaffold, TopAppBar e garantir a responsividade com verticalScroll.
